
Connectome Mapper 3
=======================================================

Connectome Mapper 3, part of the Connectome Mapping Toolkit (CMTK), implements a full diffusion MRI processing pipeline,
from raw Diffusion/T1/T2 data to multi-resolution connection matrices.
It also offers support for resting-state fMRI data processing.
